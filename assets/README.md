# Zentro Website

Static informational website for **Zentro** using the app brand color **#5F9446**.

## Pages

- `index.html` — Home
- `privacy.html` — Privacy Policy (template)
- `delete-account.html` — Account deletion instructions (for Google Play requirement)
- `terms.html` — Terms & Support (template)

## Before publishing

Search/replace these placeholders:

- `Zentro.com@gmail.com`
- Company/developer name (if different from `JSTCLICK`)

## Run locally

Open `index.html` in your browser.

## Deploy

Upload the `zentro-website/` folder to any static host:

- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting
- Any shared hosting

Then use these URLs in Play Console:

- Privacy Policy URL → `.../privacy.html`
- Delete account URL → `.../delete-account.html`
